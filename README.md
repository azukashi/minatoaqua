## Getting Started

### Clone this Repository

```sh
$ git clone https://github.com/gifaldyazkaa/aqua-fanpage.git
$ cd aqua-fanpage
```

### Install all dependencies

```sh
# Using Yarn
$ yarn install
# Using pnpm
$ pnpm install
```

### Run the app in development server

```sh
# Using yarn
$ yarn run dev
# Using dum
$ dum dev
```

### Open http://localhost:5000 to see the result.
